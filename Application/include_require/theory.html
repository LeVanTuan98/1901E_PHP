Sự khác biệt giữa Include - include_once - require - require_once

        Giống nhau:
            - Đều dùng để nạp 1 file vào 1 file khác
        Khác nhau:
            - Include: Khi quá trình nạp 1 file bị lỗi thì chương trình vẫn tiếp tục chạy
            - Require: Khi quá trình nạp 1 file bị lỗi thì chương trình sẽ dừng lại ngay
            - Include_once: Giống Include nhưng chỉ nạp file 1 lần dù file được gọi nhiều lần
            - Require_once: Giống Require nhưng chỉ nạp file 1 lần dù file được gọi nhiều lần
